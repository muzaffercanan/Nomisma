# Nomisma

## Teknik Gereksinimler

Nomisma, dijital kredi ve geri odeme sureclerini yonetmek icin hazirlanan full-stack bankacilik case-study uygulamasidir.

Teknoloji hedefleri:

- Backend: C# / .NET 8
- Frontend: React + TypeScript
- Database: Microsoft SQL Server
- Mimari: Clean Architecture

Kurulum, demo kullanicilari, finansal hesaplama modeli ve odeme kurallari uygulama tamamlandikca bu dosyada detaylandirilacaktir.
